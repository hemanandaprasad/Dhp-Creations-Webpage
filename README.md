# DHP Creations Webpage

This is a full-stack project for **DHP Creations**, an independent film production house.  
It allows users to submit applications for acting, writing, editing, directing, and cinematography roles. Submissions are stored in a **PostgreSQL database** and can be viewed on a dedicated backend page.

---

## **Project Structure**
Dhp-Creations-Webpage/
│
├── app.py # Flask backend application
├── index.html # Frontend HTML page (deployed on Netlify)
├── script.js # Frontend JS for form submission
├── style.css # Frontend CSS styles
├── requirements.txt # Python dependencies
├── Procfile # For deploying Flask app on Render
└── README.md # Project documentation


---

## **Technologies Used**

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python, Flask, Flask-CORS  
- **Database:** PostgreSQL (hosted on Render)  
- **Deployment:**  
  - **Frontend:** [Netlify](https://dhpcreations.netlify.app/)  
  - **Backend:** [Render](https://dhp-creations-webpage.onrender.com)

---

## **Features**

- Submit applications through a **friendly form**.  
- Data is stored securely in **PostgreSQL**.  
- View all submissions on a **styled table** in the backend.  
- Easy **Back to Home** navigation from backend to frontend.  
- Fully deployed and ready for demo.

---

## Deployment

- **Frontend:** Deployed on Netlify → [https://dhpcreations.netlify.app/](https://dhpcreations.netlify.app/)  
- **Backend:** Deployed on Render → [https://dhp-creations-webpage.onrender.com](https://dhp-creations-webpage.onrender.com)  
- **Database:** Render PostgreSQL

---

## Demo Flow

1. Open the **Netlify frontend** URL.  
2. Fill out the form and submit.  
3. Data is stored in the **Render PostgreSQL database**.  
4. Open **Render `/data` page** to view submissions.
